Setup Workflow

Get up and running in minutes

1

Create Account

Sign up for free and verify your email. Choose your deployment type (cloud, on-premise, or hybrid).

2

Download Edge App

Download the lightweight edge application for your device (Linux, macOS, or Windows). Minimal resource footprint (<500MB).

3

Configure Cameras

Add your cameras via IP address or RTSP streams. Supports ONVIF standard cameras and major manufacturers (Hikvision, Dahua, Uniview, etc.).

4

Define Rules

Set detection rules via the dashboard (person detection, intrusion zones, activity patterns). No coding required.

5

Enable Notifications

Configure alert channels (SMS, email, Slack, webhooks). Set up escalation and notification rules.

6

Start Monitoring

View live feeds, access dashboards, and begin receiving alerts. Start analyzing real-time and historical data.
