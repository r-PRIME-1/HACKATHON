AI Smart Curveilance

Real-time computer vision • Privacy-first • Edge-enabled

Curveilance is an AI-powered smart surveillance system that delivers low-latency, accurate detection and actionable alerts while preserving privacy. It combines state-of-the-art computer vision models, on-device inference, and secure cloud analytics to monitor environments, detect anomalies, and provide insights for safety and operations.

Request Demo Back to Home

Overview

Curveilance fuses lightweight, optimized neural networks with edge compute to perform people counting, object detection, loitering and intrusion detection, and crowd analytics in real time. The system minimizes bandwidth by sending only metadata and on-demand video clips to the cloud.

Core Capabilities

  • Computer Vision: detection, tracking, re-identification
  • Edge AI: on-device inference & accelerated pipelines
  • Analytics: real-time alerts, dashboards, historical trends
  • Privacy-First: anonymization, selective recording, secure storage

Performance

Optimized inference pipelines achieve sub-100ms detection on supported hardware and conserve energy through dynamic sampling.

Integrations

Connectors for SIEM, incident management, building management systems, and REST APIs enable easy integration into workflows.

Deployments

  • Campus Safety — zone monitoring and lost-item detection
  • Retail Analytics — footfall, dwell time, and queue management

Roadmap

Planned enhancements include multi-camera analytics, improved re-identification, and federated learning for private model updates.

Features

Lightweight Models

Compressed models deliver high accuracy with low compute requirements.

How it works

  1. Capture: Streams are ingested at the edge.
  2. Process: Optimized models run locally for fast detection.
  3. Protect: Sensitive pixels are anonymized before upload.
  4. Analyze: Selective clips and metadata are sent for aggregation.

FAQ

Is video stored in the cloud?

Only anonymized or user-approved clips are uploaded; metadata is the default payload.

What hardware is supported?

We target ARM-based edge devices and x86 gateways with optional GPU acceleration.

How is privacy handled?

Faces and identifying pixels can be blurred at the edge; retention is configurable per site.

Can I self-host?

Yes — we support on-prem deployments and hybrid cloud options for sensitive sites.
